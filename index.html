Greetings All,

Welcome to my retro version of DesignIsOrion.

I wanted to create this page in honor of the greatest gaming company ever... Nintendo. Since a youth, Nintendo was the console of many households. Starting with the iconic titles such as Super Mario Bros and Duck Hunt were the start of my gaming journey. Then it became titles such as Rygar, Contra and many others (too many to name) that had me hooked forever.


This website was created using Nes.css, an amazing CSS framework. This framework is open source and is quite light weight. With knowledge of HTML and CSS basis, you can definately use this framework with ease. Nes.css offers a variety of componets as well which you can see offer a resemblement of Nintndo entirely.


Take a moment to browse this page in enjoy the awesomeness in which it provides.
